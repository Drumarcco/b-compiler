Características

B es un lenguaje de programación desarrollado en Bell Labs en 1969 por Ken Thompson y Dennis Ritchie. B es una derivación de BCPL (Basic Combined Programming Language), y su nombre puede ser una abreviación de BCPL.

B fue diseñado para aplicaciones recursivas, no numéricas, independientes de maquina, tales como software de sistemas y lenguaje.

Thompson inventó la asignación aritmética para B, usando por ejemplo "x =+ y" para sumar 'y' a 'x'.

B no tiene tipado, o más precisamente, solo tiene un tipo de dato: la palabra de computadora.

Análisis Sintáctico

BNF


program ::=
    {definition}0

definition ::=
    name {[ {constant}01 ]}01 {ival {, ival}0 }01 ;
    name ( {name {, name}0 }01 ) statement

ival ::=
    constant
    name

statement ::=
    auto name {constant}01 {, name {constant}01 }0 ; statement
    extrn name {, name}0 ; statement
    name : statement
    case constant : statement
    { {statement}0 }
    if ( rvalue ) statement {else statement}01
    while ( rvalue ) statement
    switch rvalue statement
    goto rvalue ;
    return {( rvalue )}01 ;
    {rvalue}01 ;

rvalue ::=
    ( rvalue )
    lvalue
    constant
    lvalue assign rvalue
    inc-dec lvalue
    lvalue inc-dec
    unary rvalue
    & lvalue
    rvalue binary rvalue
    rvalue ? rvalue : rvalue
    rvalue ( {rvalue {, rvalue}0 }01 )

assign ::=
    = {binary}01

inc-dec ::=
    ++
    --

unary ::=
    -
    !

binary ::=
    |
    &
    ==
    !=
    <
    <=
    >>
    >=
    <<
    >>
    -
    +
    %
    *
    /

lvalue ::=
    name
    * rvalue
    rvalue [ rvalue ]

constant ::=
    {digit}1
    ' {char}12 '
    " {char}0 "

name ::=
    alpha {alpha-digit}0

alpha-digit ::=
    alpha
    digit
